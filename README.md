# MyUnitConverter

MyUnitConverter is an application for iOS platform that converts unit for basic measurements such as temperature and weight.

Start Date: Tue Sep 5 2017

## User Stories

The following functionality is included:

* [ ] User can enter a measurement, choose a conversion unit, and see the equivalent values.

The following **optional** features are implemented:
* [ ] Settings page to change the default tip percentage.
* [ ] UI animations
* [ ] Remembering the unit measurement across app restarts (if <10mins)
* [ ] Using locale-specific languages
* [ ] Making sure the keyboard is always visible and the unit measurement amount is always the first responder. This way the user doesn't have to tap anywhere to use this app. Just launch the app and start typing.
